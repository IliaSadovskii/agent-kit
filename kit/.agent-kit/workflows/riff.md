# Riff workflow

Standalone, interactive strategic product/business brainstorm. It builds no code, creates no
branch, and never starts `ship`. Its only possible artifact is roadmap bullets added with explicit
user approval.

1. Read `.agent-kit/project/manifest.yml`, registered idea/roadmap sources, recent history, and current
   product state.
2. Use arguments as the theme; otherwise ask what product area to explore.
3. Run canonical skill `riff`: diverge widely, spar honestly, converge into keep/park/kill.
4. With explicit approval, append KEEP outcomes to the registered roadmap without rewriting
   existing content.
5. Stop and point to the provider-appropriate `ship` invocation for building a survivor.
